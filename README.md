TODO App
===========

A simple TODO app built on React.js and Node.js

Install
-------


Install dependencies:

`npm i`

Build:

`npm run build`


Run
---

```
npm start
```

To create test db, run:

`npm run initDb`

**Note:** make sure db settings are correct in `config.json`

Test
----

```
npm test
```

To create test db, run:

```
npm run initDb:test
```

**Note:** make sure db settings are correct in `config.test.json`
